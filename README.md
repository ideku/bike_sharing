## Setup Environment - Anaconda
```
conda create --name main-ds python=3.10.12
```

## Setup Environment - Shell/Terminal
```
mkdir bike_sharing
cd bike_sharing
pipenv install
pipenv shell
pip install -r requirements.txt
```

## Run steamlit app
```
streamlit run dashboard.py
```
